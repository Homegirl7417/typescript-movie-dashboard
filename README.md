#  emoticbox-app-store
Typescript와 React를 활용해 만든 영화 서비스입니다.

영화 목록 조회, 수정, 삭제, 추가 및 검색 서비스를 제공합니다.

## ⚙️ 설치 및 세팅
### 1. npm 설치
``` shell

```

- Prettier - Code formatter
- Style Guide: Airbnb

### 3. Scripts
